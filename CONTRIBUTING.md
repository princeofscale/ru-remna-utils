# Вклад в проект RU Remna Utils

Спасибо за интерес к проекту! Мы рады любому вкладу в развитие утилит для Remnawave Panel.

## Как внести вклад

### 1. Создание Issue
Перед началом работы над новой функциональностью или исправлением ошибки, создайте issue в репозитории. Это поможет:
- Обсудить идею с сообществом
- Избежать дублирования работы
- Получить обратную связь

### 2. Разработка
1. **Fork** репозитория
2. Создайте **новую ветку** для вашей работы:
   ```bash
   git checkout -b feature/your-feature-name
   ```
3. Внесите изменения и протестируйте их
4. Убедитесь, что код соответствует требованиям (см. ниже)

### 3. Pull Request
1. Убедитесь, что ваша ветка обновлена с main
2. Создайте pull request с подробным описанием изменений
3. Дождитесь ревью и внесите необходимые правки

## Требования к коду

### Общие правила
- Код должен быть совместимым с Python 3.7+
- Поддержка Windows/macOS/Linux где возможно
- Документация на русском языке
- Использование type hints где возможно
- Читаемый и поддерживаемый код

### Структура утилит
Каждая утилита должна иметь:
- Собственную папку в корне репозитория
- README.md с инструкциями по установке и использованию
- Требования в requirements.txt или pyproject.toml
- Тесты в папке tests/

### Стиль кода
- Следуйте PEP 8
- Используйте black для форматирования
- Добавляйте docstrings для функций и классов
